TiPortfolio allows you to use in AI Agents too as a skill.


You can use it Claude.code or Cursor to ask Agents to help you run backtests and get insights from the results.

It will connect different services and different tools to provide accurate portfolio asset allocation backtest results and insights.


For example


```markdown

Backtest a strategy that invests 50% in SPY, 30% in QQQ, and 20% in GLD, rebalancing at the end of each month, from January 1, 2019 to December 31, 2024. Provide a summary of the results.
```
